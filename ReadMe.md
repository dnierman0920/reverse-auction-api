# Reverse Auction API

- ## **Summary:** An Api that tracks projects, bids, vendors and buyers.

- ## **What is a reverse auction?** - multiple sellers that compete for the business of a single buyer.

&nbsp;

## Types of objects:

    1. Sellers
    2. Buyers
    3. Bids
